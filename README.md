# Progetto Calcolatore Interesse Semplice

Questo repository contiene uno script Bash creato per calcolare l'interesse semplice. 
Il progetto è parte del corso "Introduction to Git and GitHub".

## Descrizione
Lo script `simple-interest.sh` calcola l'interesse basandosi su tre input forniti dall'utente:
1. **Capitale** (Principal)
2. **Tasso di interesse** annuo (Interest Rate)
3. **Periodo di tempo** in anni (Time period)

## Formula utilizzata
La formula per il calcolo è:
I = (P * R * T) / 100
